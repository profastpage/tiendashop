# TechStore Perú - Tienda Online 🇵🇪

Una tienda online moderna y responsiva de productos Apple y tecnología premium.

## 🚀 Características

- **Diseño Moderno** - UI/UX profesional con TailwindCSS
- **Modo Oscuro** - Toggle para cambiar entre tema claro/oscuro
- **100% Responsivo** - Funciona en todos los dispositivos
- **Carrito de Compras** - Gestión completa con localStorage
- **Filtros Avanzados** - Búsqueda, categoría, precio, rating
- **Wishlist** - Lista de favoritos persistente
- **Comparador** - Compara hasta 3 productos
- **Vista Rápida** - Modal ligero de productos
- **Newsletter** - Popup con código de descuento
- **Badges** - Nuevo, Oferta, Más Vendido
- **Countdown** - Temporizador de ofertas flash
- **Zoom Imágenes** - Efecto zoom en detalle de producto
- **Scroll Animations** - Animaciones al hacer scroll
- **SEO Optimizado** - Meta tags, Open Graph, Twitter Cards
- **Accesibilidad** - ARIA labels, navegación por teclado
- **Google Analytics** - Integración lista

## 🛠️ Tecnologías

- HTML5
- CSS3 (TailwindCSS)
- JavaScript (Vanilla)
- Remix Icons
- Google Fonts (Inter)

## 📦 Instalación

1. Clona este repositorio
2. Abre `index.html` en tu navegador

```bash
# O usa un servidor local
npx serve
```

## 🎨 Personalización

- **Colores**: Edita las variables CSS en `<style>`
- **Productos**: Modifica el array `products` en el JavaScript
- **WhatsApp**: Cambia `WHATSAPP_NUMBER` en la configuración
- **Analytics**: Reemplaza `G-XXXXXXXXXX` con tu ID de Google Analytics
